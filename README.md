### About

This is an implementation of a simple thread pool to explore concurrency in
modern C++.

### Build

1. mkdir build && cd build
2. cmake ..
3. cmake --build .


### Run example

    ./threadPoolExample


### References

* C++ Concurrency in Action, 2ed, Anthony Williams, 2019
